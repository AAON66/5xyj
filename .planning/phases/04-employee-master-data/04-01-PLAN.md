**4. employee_service.py: 导入容错模式 (per D-07)**

当前 `_parse_employee_rows`（或等价的循环函数）中，`_parse_employee_row` 对缺少必填字段抛 `EmployeeImportError` 导致整个导入中断。改为在循环中 catch 并收集：

```python
def _parse_employee_rows(file_name: str, raw_bytes: bytes) -> tuple[list[_EmployeeImportRow], list[str]]:
    # ... 现有 dataframe 准备逻辑不变 ...
    rows: list[_EmployeeImportRow] = []
    errors: list[str] = []
    for offset, row in enumerate(dataframe.to_dict(orient="records"), start=2):
        try:
            parsed = _parse_employee_row(row, column_map, row_number=offset)
        except EmployeeImportError as exc:
            errors.append(str(exc))
            continue
        if parsed is None:
            continue
        rows.append(parsed)
    return rows, errors
```

注意：`_parse_employee_row` 本身保持抛异常不变，容错逻辑在调用处处理。

  <action>
  1) 创建 scripts/check-hardcoded-colors.sh（白名单排除 theme/index.ts、semanticColors.ts、chartColors.ts、MainLayout.module.css、index.html）:
  ```bash
  #!/bin/bash
  set -e
  cd "$(dirname "$0")/.."
  VIOLATIONS=$(grep -rEn "style=\\{\\{[^}]*#[0-9a-fA-F]{3,8}" frontend/src/ \
    --include="*.tsx" --include="*.ts" --include="*.jsx" --include="*.js" \
    --exclude-dir=node_modules --exclude-dir=dist 2>/dev/null \
    | grep -v -E "(theme/index\\.ts|semanticColors\\.ts|chartColors\\.ts|MainLayout\\.module\\.css|index\\.html)" || true)
  if [ -n "$VIOLATIONS" ]; then
    echo "FAIL: Hardcoded inline-style colors found:"
    echo "$VIOLATIONS"
    exit 1
  fi
  echo "PASS: No hardcoded inline-style colors outside whitelist."
  ```

  2) 执行：`chmod +x scripts/check-hardcoded-colors.sh && bash scripts/check-hardcoded-colors.sh`

  3) 补充 borderColor/background/color 单属性 grep 验证：
  ```
  grep -rEn "(borderColor|background|color): *'#[0-9a-fA-F]{3,8}'" frontend/src/ --include="*.tsx" 2>/dev/null \
    | grep -v -E "(theme/index\\.ts|semanticColors\\.ts|chartColors\\.ts)"
  ```
  必须为空（或仅剩 Sider 深色 `#1F2329` 在允许列表中）。
  </action>

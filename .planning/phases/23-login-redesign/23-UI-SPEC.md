## Color

Base palette inherits `theme/index.ts` + `semanticColors.ts`. The login page MUST NOT introduce new hex values outside the list below.

### Light Mode

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#F5F6F7` (`colorBgLayout`) | Right column background, page fallback background |
| Secondary (30%) | `#FFFFFF` (`colorBgContainer`) | Form card surface, input backgrounds |
| Accent (10%) | `#3370FF` (`colorPrimary` / BRAND) | Particle wave gradient base, primary "登录" / "验证身份" CTA, active tab underline, "飞书登录" icon, link text |
| Destructive | `#F54A45` (`colorError`) | Form validation error text, "登录失败" `message.error` tint (Ant Design default) |

Canvas gradient (light mode): linear gradient from `#E8F0FF` (particle bright) → `#3370FF` (particle mid) → `#1D4AC7` (particle deep) for the wave color ramp. These are **particle colors only**, never applied to surfaces or text.

### Dark Mode (D-06, D-07)

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#141414` (Ant dark algorithm layout bg) | Right column background behind the glass card |
| Secondary (30%) | `rgba(30, 35, 45, 0.72)` | Form card surface (translucent, uses `backdrop-filter: blur(20px) saturate(180%)`) |
| Accent (10%) | `#3370FF` base + `#7FA7FF` glow | Particle "deep-sea luminescence" — brighter particles on dark canvas, CTA button keeps brand blue |
| Destructive | `#F54A45` | Form validation error text |

Canvas gradient (dark mode): linear gradient from `#7FA7FF` (glow bright) → `#3370FF` (mid) → `#1B2B4D` (deep) with additive blending for the "deep-sea luminescence" effect (D-06).

### Accent Reserved For (explicit list — NEVER applied elsewhere)

1. The Three.js particle wave gradient (`#3370FF` family) on the left canvas.
2. Primary submit button background: `Button type="primary"` (登录 / 验证身份).
3. Active tab indicator (Ant Design `Tabs` built-in primary tint).
4. `<Link>` text color for "进入员工查询".
5. The 飞书 OAuth button icon (`ApiOutlined`) — NOT the button background (button stays default/ghost).
6. Form field focus ring (Ant Design default `colorPrimary` focus outline).

Accent is **NOT** used for: card borders, dividers, tab body backgrounds, slogan text, regular copy, hover states on secondary buttons.

### WebGL Fallback Color (D-08)

When WebGL unavailable, left canvas renders a CSS linear gradient with slow drift:
- Light: `linear-gradient(135deg, #E8F0FF 0%, #3370FF 60%, #1D4AC7 100%)` with 20s `backgroundPosition` animation.
- Dark: `linear-gradient(135deg, #1B2B4D 0%, #3370FF 60%, #7FA7FF 100%)` same animation.

## 3D Canvas Contract

Prescriptive parameters for the Three.js particle wave component. Executor uses these values directly.

| Parameter | Value | Source |
|-----------|-------|--------|
| Particle count | **2500** (inside the 2000-3000 band from D-02) | D-02 |
| Particle shape | Circular point sprite (`THREE.Points` with soft circle texture, no square points) | D-05 |
| Particle size | 2.0px base, 3.0px on crest | Discretion |
| Grid layout | Plane grid 80 × 32 (= 2560 ≈ 2500 after edge trim), spaced 0.4 world units | Discretion |
| Wave equation | `z = sin(x * freq + t * speed) * amp + sin(y * freq2 + t * speed2) * amp2` — dual sine for organic flow | D-01 |
| Frequency x | 0.35 | Discretion |
| Frequency y | 0.25 | Discretion |
| Base amplitude | 0.6 world units | Discretion |
| Animation speed | 0.6 rad/s | Discretion |
| Mouse interaction | Radial "gaussian bulge" centered on pointer, sigma 3.5 units, amplitude +1.2 units, decay 0.8s after pointer leaves | D-03 |
| Color mapping | Per-vertex color lerp between 3 stops along the Z height (deep → mid → bright) | D-04 |
| Camera | Perspective, fov 55, pos (0, -14, 12), lookAt (0, 0, 0) | Discretion |
| Renderer | `WebGLRenderer({ antialias: true, alpha: true })`, `setPixelRatio(Math.min(2, devicePixelRatio))` | Discretion |
| Frame budget | Target ≤ 8ms per frame; pause `requestAnimationFrame` when `document.hidden` | Discretion |
| Lazy load | `React.lazy(() => import('./ParticleWave'))` + `<Suspense fallback={<CssGradientBackground />}>` | D-09 + discretion |

**WebGL detection (D-09):**
- On mount, attempt `canvas.getContext('webgl2') ?? canvas.getContext('webgl')`.
- If null, unmount the 3D subtree and render `<CssGradientBackground />` (see WebGL Fallback Color above).
- Also fall back if `renderer.getContext().isContextLost()` fires.

**Reduced motion:**
- If `window.matchMedia('(prefers-reduced-motion: reduce)').matches`, skip the wave animation and mouse interaction; render a static frozen particle grid (still in brand colors).

## Interaction & Motion Contract

| Interaction | Behavior |
|-------------|----------|
| Form card mount | Fade-in + 8px upward translate, 300ms `cubic-bezier(0.645, 0.045, 0.355, 1)` (matches `motionEaseInOut` token) |
| Canvas mount | No entrance animation; particles start in neutral plane, wave animation begins immediately |
| Mouse move over left canvas | Gaussian bulge follows pointer within 80ms; returns to baseline over 800ms after `mouseleave` (D-03) |
| Tab switch | Ant Design default (no custom motion) |
| Form submit | Button `loading` state, full-width block button (preserved) |
| Theme toggle (if triggered while on login) | Particle color ramp + card background cross-fade 200ms (uses `motionDurationMid`) |
| Candidate modal open | Ant Design default (`CandidateSelectModal` unchanged) |

**Reduced motion:** When `prefers-reduced-motion: reduce` is set — skip card fade-in, skip canvas animation (static grid), keep functional transitions (button loading, modal open) at 0ms opacity flip.

## Accessibility Contract

| Rule | Value |

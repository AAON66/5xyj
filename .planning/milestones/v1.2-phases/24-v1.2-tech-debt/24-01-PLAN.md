**Backend API（线上实现，任务必须对齐）：**
```
GET  /api/v1/auth/feishu/authorize-url
  -> 200 { success: true, data: { url: string, state_signed: string } }
  URL 形如 https://accounts.feishu.cn/open-apis/authen/v1/authorize?client_id=...&state=...
  state_signed 是 `{state}.{HMAC(state, auth_secret_key)}` 格式

POST /api/v1/auth/feishu/callback
  body: { code: string, state: string, state_signed: string }

  成功：调 _fetch_feishu_user_info(code, settings) 拿 open_id/union_id/name，走三级匹配

```

hook 签名：`export function useMenuOpenKeys(defaultKeys: string[], validKeys: string[]): { openKeys: string[]; onOpenChange: (keys: string[]) => void }`

实现细节：
1. `const STORAGE_KEY = 'menu-open-keys';`（per D-10）
2. useState 初始化函数中：
   - 尝试从 localStorage 读取并 JSON.parse
   - 验证结果是 Array（`Array.isArray(parsed)`）
   - 用 `validKeys` 参数过滤掉不再存在的 group key（addresses review concern: 角色切换或飞书开关变化后旧值包含无效分组）：`parsed.filter((k: unknown) => typeof k === 'string' && validKeys.includes(k))`
   - 成功则使用过滤后的数组，失败（包括 JSON 解析错误、非数组、window 不存在）回退到 `defaultKeys` 参数
3. `onOpenChange` 回调：调用 setOpenKeys 更新 state，同时 try/catch 写入 localStorage（per D-11）
4. 使用 `useCallback` 包裹 onOpenChange 以保持引用稳定

完整实现：

```typescript
import { useState, useCallback } from 'react';

const STORAGE_KEY = 'menu-open-keys';

export function useMenuOpenKeys(defaultKeys: string[], validKeys: string[]) {
  const [openKeys, setOpenKeys] = useState<string[]>(() => {
    try {
      const saved = localStorage.getItem(STORAGE_KEY);
      if (saved) {
        const parsed = JSON.parse(saved);
        if (Array.isArray(parsed)) {
          const cleaned = parsed.filter(
            (k: unknown) => typeof k === 'string' && validKeys.includes(k)
          );
          return cleaned.length > 0 ? cleaned : defaultKeys;
        }
      }
    } catch {
      /* fallback to default — covers JSON parse error, Safari private mode, SSR */
    }
    return defaultKeys;
  });

  const onOpenChange = useCallback((keys: string[]) => {
    setOpenKeys(keys);
    try {
      localStorage.setItem(STORAGE_KEY, JSON.stringify(keys));
    } catch {
      /* Safari private mode or quota exceeded */
    }
  }, []);

  return { openKeys, onOpenChange };
}
```

```typescript
/** Find which group contains the given menu key, used to auto-expand parent group */
function findParentGroupKey(menuKey: string, groups: MenuGroupConfig[], feishuItems: NavItem[]): string | null {
  for (const group of groups) {
    const allChildren = group.key === 'group-admin'
      ? [...group.children, ...feishuItems]
      : group.children;
    if (allChildren.some(child => child.key === menuKey)) {
```

**步骤 5：重写 buildMenuItems 函数**

```typescript
function buildMenuItems(
  topItem: NavItem,
  groups: MenuGroupConfig[],
  feishuItems: NavItem[],
  userRole: string
): MenuProps['items'] {
  // 员工角色（per IC-07）：不分组，只返回 employee 可见的单项
  if (userRole === 'employee') {
    return [{ key: '/employee/query', icon: <SearchOutlined />, label: '员工查询' }];
  }

  const items: MenuProps['items'] = [];

  // 置顶项：快速融合（per D-02）
  if (topItem.roles.includes(userRole)) {
    items.push({ key: topItem.key, icon: topItem.icon, label: topItem.label });
  }

  // 各分组 SubMenu
  for (const group of groups) {
    let groupChildren = [...group.children];
    // 管理组插入飞书菜单项
    if (group.key === 'group-admin' && feishuItems.length > 0) {
      groupChildren = [...groupChildren, ...feishuItems];
    }

    const visibleChildren = groupChildren
      .filter(child => child.roles.includes(userRole))
      .map(child => ({ key: child.key, icon: child.icon, label: child.label }));

    if (visibleChildren.length > 0) {
      items.push({
        key: group.key,
        icon: group.icon,
        label: group.label,
        children: visibleChildren,
      });
    }
  }

  return items;
}
```

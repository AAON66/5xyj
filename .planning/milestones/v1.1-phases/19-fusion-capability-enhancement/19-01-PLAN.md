<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Rule CRUD API | 用户通过 API 写入可影响后续导出的覆盖规则，必须限制字段与权限 |
| Employee targeting | 规则一旦命中错误员工，会导致导出金额被篡改 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-19-01 | Tampering | fusion-rules CRUD | mitigate | 只允许 `personal_social_burden` / `personal_housing_burden` 两个 field_name，通过 schema 与 service 双重白名单校验 |
| T-19-02 | Elevation of Privilege | fusion-rules API | mitigate | `employee` 角色返回 403，仅 `admin/hr` 可访问 |
| T-19-03 | Spoofing | employee scope match | mitigate | 规则仅允许 `employee_id` / `id_number` 两类 scope_type，不允许仅按姓名 |
</threat_model>
